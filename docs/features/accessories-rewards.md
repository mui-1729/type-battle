# アクセサリー・ポイント・ローカル保存

## 保存

プレイヤー設定は `type-battle:settings`、報酬状態は `type-battle:rewards:v1` としてブラウザの `localStorage` に保存する。報酬データはバージョン番号を持ち、破損・不正なJSON・未知の値は初期状態へ安全にフォールバックする。

## アクセサリー

初期アクセサリーは「なし」「キャップ」「ハチマキ」。5ポイントでサングラス・ヘッドホン・魔法使いの帽子、10ポイントで王冠・武将風のかぶと・天使の輪を解放する。アクセサリーは見た目だけを変更し、速度、正確率、攻撃力、HPへ影響しない。

## 報酬ルール

- 公式プリセットで正確率100%、ミス0の完走をした場合だけ対象にする
- ルーム対戦・COM戦はルールごとに1日1ポイントまで
- デイリーチャレンジは日付ごとに1ポイントまで
- 練習、CUSTOM設定、棄権勝ちは対象外
- 報酬キーを保存して、リロードや再戦での重複付与を防ぐ

判定はクライアントの表示ではなく、Workerから返された試合結果のプレイヤー統計を使う。将来アカウント保存へ移行する場合は `player-rewards.ts` の保存境界を置き換える。
